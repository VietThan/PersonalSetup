## Install `code` command in PATH

Can one really call themselves a programmer if they can't open applications on the command line?

1. `⌘ Cmd` + `⇧ Shift` + `p`
2. Search for **Shell Command: Install `code` command in PATH**
3. Now go enjoy:

```{ .console .copy }
code .
```
